## ü§ì Introduction

Trello is a clone of the real Trello, offering the same main functionalities as the original platform.

## ü§© Features

1. Login/Register with JWT token authentication
2. Ability to create/update/delete the board
3. Ability to add/update/move/delete the card
4. Background image library for the board
5. Add labels to the card
6. Supports adding of detail description in the card
7. Assign a card to the user

## ‚õìÔ∏è Installation
First install the `MongoDB Compass` for better visualization of data with `MongoDB` server.

1. Clone and install the dependencies for `Trello` locally:

```tf
  git clone git clone https://github.com/S-Tentacion/TRELLO.git
  cd MOVIE_WIRE
  npm install
```
2. Create `.env.local` and add this env variable `LOCAL_MONGODB=mongodb://localhost:27017/trello` Add `JWT_SECRET_KEY=randomstrings`
3. Run `yarn install`
4. Run `yarn dev`
5. `For unsplash gallery, api key is needed which can be generated from unsplash website`
