## ü§ì Introduction

This project involves drone navigation, where users select a starting point and a destination. The drone then displays path to reach the chosen destination from the selected starting point.

## ü§© Features

1. Google Maps integration in React.
2. Ability to add markers on the map.
3. Panning and zooming functionality for the maps.
4. Option to disable default map controls.
5. Places Autocomplete for easy location input.
6. Directions Service to calculate routes between locations.
7. Directions Renderer to display route information on the map.
